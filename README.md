# Branding

Treasure brand icons

## Installation

```bash
npm install @treasure-project/branding
```

## Usage

```tsx
import {
  TreasureLogo,
  MagicLogo,
  TreasureLogoFull,
} from "@treasure-project/branding";

// Use it in React
```
